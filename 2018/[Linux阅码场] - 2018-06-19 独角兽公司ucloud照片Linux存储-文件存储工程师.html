高性能分布式存储研发工程师(块存储/文件存储)   

岗位描述：

1. 我们负责分布式块存储，分布式文件存储的研发和运营。为存储打造极致的性能是我们的使命，永远追求更低的延迟，更高的IOPS，更高的带宽。

2. 我们致力于研发和打造高可靠，高可用的存储系统，保障数据安全是我们义不容辞的责任；并提升存储利用率，降低用户的存储成本；

3. 我们主要承担云主机的块设备的存储需求，以及 docker，rds，分布式数据库，机器学习等的业务存储需求；

4. 积极跟进存储行业的发展，及时引入新型硬件( nvme , rdma网卡 等), 充分理解计算机体系结构，并引入kernel bypass 软件栈（ dpdk, sdpk 等）来充分发挥硬件的性能；

5. 发现，分析并解决分布式存储业务里网络性能瓶颈，存储性能瓶颈，并更好的进行线上运营，提高运营质量和效率，更好的进行灰度发布控制，保障业务质量；

6. 日常自动化运维体系的建设，减少人工处理，通过代码减少重复劳动；


岗位要求：

1. 热爱技术，追求代码的高质量，追求技术的极致，愿意写出充分发挥硬件性能的软件；

2. 积极主动，敢于承担任务，敢于承担责任；

3. 熟悉 Linux 系统下 C/C++ 开发；

4. 熟悉 Linux 系统下 网络编程；

5. 熟悉业界常见的存储系统较佳，如 Ceph, GlusterFs, Leveldb, Redis, Mongodb，Mysql 等；



具备以下能力加分：

1. 熟悉 Linux 系统下存储相关知识，IO 知识及优化 较佳；

2. 熟悉 kernel bypass（ RDMA, DPDK, SPDK ）相关知识 较佳；

3. 熟悉 linux 内核，IO 栈，网络栈等较佳;

4. 熟悉 Paxos, Raft, 2PC, 3PC, Crush，一致性 Hash, Qos, ISCSI, NBD, NAS 等技术与理论较佳；

5. 不具备以上条件也没问题，我们欢迎自信的人一起打造一流的分布式存储系统。
