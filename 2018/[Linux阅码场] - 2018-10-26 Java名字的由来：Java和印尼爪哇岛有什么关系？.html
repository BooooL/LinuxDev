既然没人回答，我说一说，根据真实故事添油加醋。

大概就像NBA巨星科比，和神户牛排的那种关系吧（这个不懂的百度就有）。所以严格来说，java编程语言和爪哇岛还是有那么点联系的（这个不懂百度也有，但含糊不清，有兴趣往下看看）。

印尼爪哇岛火山

最早这个编程语言被叫做Oak，申请注册的时候被告知这个名字已经被注册了（又一个同名栗子），发明者吐血不止，但先来后到的关系，他无可奈何。

然后他想想用Oak233，Oak666这种带数字后缀的很他妹low，就擦干嘴角的鲜血回去召集了一个改名会议。

大伙儿知道不能用Oak后，都很沮丧，但名字还是要起，于是大会上争吵不休，员工们各抒己见，最后从无数逗比的名字中筛选出了三个名字：Silk， DNA， Java。接着所有人对这三个名字进行投票，而DNA同时获得了最多票数的“最受欢迎票”和“最不受欢迎票”，被弃用（这个要是被采用，以后叫programming in DNA，感觉很恶心。）。剩下的两个中，Java获得了大部分支持票数，于是被最终采用。

为什么会出现Java这个名字呢？参与新名称提议的与会者们在会上都喝一款叫Peet's Java的咖啡（加州产），有人就顺口喊出Java这个名字了。据说是一个名叫Chris Warth的工程师首先提出的，但他本人并不认为，他说当时会场一片混乱，天杀的知道是谁第一个说的。不过，谁说的和本题有关系？



总而言之，就是会议上随性喊出来的并被最终采用的名字。和爪哇岛有没关系呢？有那么一点点吧，那款咖啡叫做Peet's Java，当然是因为其原料咖啡豆是著名咖啡豆出产地爪哇岛出产的，也就是说，编程语言Java的名字就是起源于印尼小岛Java的名字，属于第八杆子的尖刚好打到的那种关系。
