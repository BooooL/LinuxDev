fuck

当你把clear命令，敲成了clare，然后系统提示：

baohua@baohua-VirtualBox:~$ clare

No command 'clare' found, did you mean:

 Command 'flare' from package 'flare' (universe)

 Command 'glare' from package 'radiance' (universe)

clare: command not found

此时，Ubuntu提示你是不是敲“flare”，或者“glare”，你心中一万个草泥马奔涌而过的时候，你再敲一个fuck：

baohua@baohua-VirtualBox:~$ fuck

clear [enter/↑/↓/ctrl+c]

fuck，真体贴，它知道你要敲的是clear，简单回车，即可执行clear。所以，fuck命令，帮你自动纠正前一条命令的错误。

sl

sl是最经典的ls的反义词。

#apt-get install sl后，当你敲下sl命令，你会看到一个小火车，从你屏幕缓缓驰过....那是你的童年的美好的回忆。


下面我们把fuck和sl结合一下：

baohua@baohua-VirtualBox:~$ sudo apt-get instal sl

E: Invalid operation instal

baohua@baohua-VirtualBox:~$ fuck

sudo apt-get install sl [enter/↑/↓/ctrl+c]
