关于大疆

深圳市大疆创新科技有限公司（简称“大疆”），总部位于中国深圳，在北京、上海、西安设有专业研发中心。

技术创新是大疆的立足之本。从飞行控制系统起家，大疆凭借稳定高效的消费级和专业级的航拍飞行平台为人熟知，并将技术移植到地面增稳设备，实现“天地一体”的拍摄解决方案。十二年间，DJI大疆创新不断革新技术和产品，开启了全球飞行影像新时代；我们的产品应用在影视、农业、地产、新闻、消防、救援、能源、遥感测绘、野生动物保护等多个领域，重塑了人们的生产和生活方式。

嵌入式软件工程师（Linux方向）

工作职责

1. 负责Linux系统层驱动开发，应用功能开发；

2. 负责所属模块的链路、架构设计及开发编码等工作；

3. 进行软件代码维护，优化解决系统级问题。

任职要求

1. 本科及以上学历，3年以上工作经验；

2. 精通C/C++语言及调试技巧；

3. 熟悉Linux驱动开发及应用程序开发；

4. 熟悉数据结构及算法优先考虑

 

嵌入式软件系统架构师

岗位职责：

1.  负责系统需求调研、可行性分析和产品规格定义；

2.  负责系统总体方案设计和详细设计，负责技术方案评审；

3.  负责新技术和关键技术的跟踪研究；

4.  负责产品质量、性能和架构优化。

任职要求：

1.  5年以上嵌入式经验，3年以上中大型嵌入式软件系统架构设计经验；

2.  深入理解Linux操作系统；

3.  精通多任务、高实时、高可用设计；

4.  精通C/C++及数据结构；

 

以上岗位工作地点可选择深圳、上海、西安。
